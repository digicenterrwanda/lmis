.. container:: table-row

  Property
    <for each="data#myData">...</for>
    
  Data type
    Object
     
  Description
    Repeats the sequence inside the tag for each element provided by means of the "each"
    attribute.
    
    Attributes\:
    
    - each (required)\: the element for the repetition.
